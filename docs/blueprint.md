# **App Name**: AgriAssist

## Core Features:

- Crop Disease Detection: AI-powered tool to identify crop diseases from images and suggest treatments. The tool will utilize Google Vision to determine when it needs to incorporate information about potential diseases.
- AI Crop Recommendations: Provide crop suggestions based on the user's location, soil composition, local climate, and market demand.
- User Profile Management: Simple profile management including photo uploads and account details.
- Weather Intelligence Display: Display hyperlocal weather forecasts relevant to the user's location.

## Style Guidelines:

- Primary color: Earthy Green (#388E3C) for representing growth and nature.
- Secondary color: Light Beige (#F5F5DC) for a clean and neutral background.
- Accent: Sunny Yellow (#FFEB3B) for highlighting important information and CTAs.
- Clean and modern typography for easy readability.
- Use agriculture-themed icons for key features.
- Clean and intuitive layout for easy navigation.
